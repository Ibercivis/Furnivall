Furnivall by Ibercivis

Furnivall is an open source framework for distributed volunteer science, mainly centered in distributed thinking. Allows researchers to take advantage of volunteer's capabilities for tasks that computers will bemuch less eficient, giving at the same time the oportunity to actively collaborate with science projects. Written in python, and using the non-blocking open source web framework "tornado", aims to be the best approach to minimize researchers effort for a collaborative thinking project, allowing them to concentrate in what matters, and leave the housekeeping to Furnivall.
