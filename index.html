Furnivall by Ibercivis

Open-source software framework for distributed thinking.
Helps to organize batches of tasks, collect them form volunteers and it does all the related housekeeping.
Currently Furnivall consists on a series of plugins + views and an integrated webserver, using threadspoolexecution to make tasks asynchronous.
