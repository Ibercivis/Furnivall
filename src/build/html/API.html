do_management(slug)[source]¶

Object manager get function. Creates as requested jobs or views Implement this user permissions on the ACL:

own_job create_job view_job own_task create_task view_task own_workunit create_workunit view_workunit root

If we want a task to create another task, the assigned user_ has to have own_task permissions! Note: For a task to create another task, I’d prefer it to be done via the web interface. We can do it at plugin level, but that would mean we wouldn’t have such a nice access to authentication methods.

produce_workunits(number=1)[source]¶

Creates N new workunit objects, with this job as job and appends them to our workunits queque. If N is 1, it will also return the created workunit.

>>> a=job(viewtest())
>>> len(a.workunits)
10
>>> a.produce_workunits(10)
>>> len(a.workunits)
20
>>> print a.workunits[0] 
<WorkUnit.workunit object at 0x...>
>>> print a.workunits[0].tasks 
deque([[0, <Assignment.task object at 0x...>]])
